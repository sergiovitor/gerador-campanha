# 🖼️ Gerador de Campanhas

> Ferramenta pedagógica desenvolvida para o letramento digital e o exercício da cidadania, permitindo que alunos criem materiais de campanha de forma simplificada e com alta qualidade visual.

Este aplicativo foi projetado para o ambiente escolar, oferecendo uma interface limpa e intuitiva que facilita a manipulação de elementos gráficos por estudantes dos anos iniciais do Ensino Fundamental. O foco é a autonomia do aluno na produção de conteúdo autoral.

---

## ✨ Funcionalidades

- **🎨 Interface Intuitiva:** Design focado na redução da carga cognitiva, ideal para o público infantil e estudantes em processo de alfabetização.
- **🖼️ Manipulação de Imagens:** Sistema simples para importação e posicionamento de fotografias dos candidatos.
- **🔠 Edição Tipográfica:** Biblioteca de estilos de fontes selecionados para garantir legibilidade e impacto visual.
- **🔒 Estabilidade de Edição:** Recurso de bloqueio de elementos de fundo para evitar alterações acidentais durante o processo criativo.
- **📥 Exportação Direta:** Geração de arquivos de imagem prontos para impressão ou divulgação em redes sociais escolares.

## 🛠️ Tecnologias

O projeto utiliza o **Tauri v2** para oferecer uma experiência nativa leve e segura:

- **Core:** [Rust](https://www.rust-lang.org/) (Performance e segurança de memória).
- **Frontend:** HTML5, CSS3 e JavaScript Vanilla.
- **Desktop:** [Tauri](https://tauri.app/) (Integração com Windows e Linux).
- **CI/CD:** GitHub Actions para compilação automatizada dos instaladores.

## 🚀 Como Executar Localmente

Caso deseje testar o código ou contribuir:

1. **Pré-requisitos:** Ter o [Rust](https://www.rust-lang.org/tools/install) e o [Node.js](https://nodejs.org/) instalados.
2. **Clonar o projeto:**
   ```bash
   git clone https://github.com/sergiovitor/gerador-campanha.git
   ```
3. **Instalar dependências:**
   ```bash
   npm install
   ```
4. **Iniciar em modo de desenvolvimento:**
   ```bash
   npm run tauri dev
   ```
